
PROYECTO HARRIS BENEDICT

La idea de este proyecto es poner en practica algunos conocimientos basicos de web (HTML,CSS y Javascript).
Tambien aprender lo basico del preprocesador de css Sass.

Lo realizado:

1 HTML
	1.1 html semantico
		<header> para el titulo de la pagina
		<nav> para la barra de navegacion
		<main> para el contenido
                <footer> para el pie de pagina 
	1.2 input
		number -> para altura, peso y edad
		radio -> para genero
   		* el uso de un section con options para la actividad semanal
2 Javascript
	2.1 se trajo los valores de los input para asi poder realizar los calculos
	2.2 verificacion de valores:
		NUMEROS no acepta numeros negativos, tampoco extraordinarios posibles para que en caso de notarse un error como un numero negativo 
			Sale en pantalla la notificacion del campo incorrecto o vacio
3 Sass
	3.1 variables para los colores
	3.2 imagen adaptada al fondo
	3.3 box-shadow para estetica y transparencia para los elementos
	3.4 estetica de los input, radio buttons con iconos
	3.5 pseudo clases css 
		:active   :checked    :hover
	3.6 flex para que sea responsive
