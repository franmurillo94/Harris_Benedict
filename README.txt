
	PROYECTO HARRIS BENEDICT

	La idea de este proyecto es poner en practica algunos
	conocimientos basicos de web (HTML,CSS y Javascript).
	Tambien aprender lo basico del preprocesador de css Sass.

	Lo que se realizo:

	1 HTML
		1.1 se planteo una pagina usando html semantico 
		    usando los tags.
		    <header> para el titulo de la pagina
		    <nav> para la barra de navegacion
		    <main> para el contenido
                    <footer> para el pie de pagina 
		1.2 el uso de input
		    number -> para altura, peso y edad
		    radio -> para sexo
   		    el uso de un section con options
		    para la actividad semanal
	2 Javascript
		2.1 se trajo los valores de los input para asi
		    poder realizar los calculos
		2.2 se verifico que los numeros fueran posibles
		    para que en caso de notarse un error como
		    un numero negativo la funcion se de cuenta
		    y lo informe. Tambien si falta un valor.
	3 Sass
		3.1 se uso variables para los colores
		3.2 se uso una imagen random de background y
		    se la adapto al fondo
		3.3 uso de box-shadow para estetica y uso de
		    la transparencia para los elementos
		3.4 se cambio la estetica de los input asi como de 
		    los radio buttons con iconos
		3.5 uso de pseudo clases css 
			:active   :checked    :hover
		3.6 se uso solamente flex para que sea responsive
